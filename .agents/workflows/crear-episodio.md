# Workflow: Crear Episodio Prisma
**Descripción:** Guía paso a paso para estructurar un episodio desde la idea hasta la publicación, forzando el sistema anti-perfeccionismo.

## Paso 1: Idea (Fase 1)
Agente, pide al usuario que defina el título tentativo, el tema, la herida central, la pregunta inicial/final y el mood sonoro. Espera su respuesta.

## Paso 2: Estructura (Fase 2)
Sugiere una división del episodio en bloques narrativos (ej. intro, apertura emocional, desarrollo 1, desarrollo 2, quiebre, cierre, reflexión final). Pide aprobación al usuario para continuar.

## Paso 3: Guion por Bloques (Fase 3)
Ayuda al usuario a redactar el texto para cada bloque. Recuérdale indicar el tono, las respiraciones y los silencios. Espera a que termine.

## Paso 4: Asignación de Sonido (Fase 4)
Sugiere audios del "Núcleo Prisma" (como *Ansiedad eléctrica*, *Confesión íntima* o *Glitch espiritual*). 
**OBLIGATORIO:** Solo puedes asignar máximo 1 drone por segmento y máximo 5 drones en todo el episodio. Asigna el volumen sugerido entre 10% y 25%. Pide confirmación.

## Paso 5: Preparar Vista de Grabación y Edición (Fase 5 y 6)
Confirma con el usuario que el guion está listo para el "modo cabina" (texto limpio, cues grandes, temporizador, checklist de tomas). Genera un borrador de la guía de montaje o lista de decisiones de edición (Edit Decision List) para la fase de postproducción.

## Paso 6: Generación de Derivados (Fase 7)
Basado en el guion final, genera automáticamente la descripción del episodio, título SEO, 1 copy para Instagram, 1 copy para TikTok y extrae 3 "quotes" (citas clave) del guion para reciclar.
